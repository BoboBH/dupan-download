## 🎯 版本概述

v1.0.6版本解决了Windows路径长度限制导致的下载失败问题，这是影响生产环境稳定性的重要修复。

## 🐛 修复问题

### 核心问题
**Windows路径长度限制**: 260字符
- **问题**: 长文件名 + 用户子目录 + 临时目录路径超过限制
- **影响**: 下载失败，文件无法找到
- **严重性**: 高 (影响长文件名处理)

### 错误示例
```
[ERROR] Failed to find downloaded file: [WinError 3] 系统找不到指定的路径
[ERROR] Download command succeeded but file not found in temp
```

### 问题文件
`Goldman Sachs-EM WEEKLY FUND FLOWS MONITOR：Another week of large FII selling， led by North Asia as Momentum unwind continues for the third consecutive week； Strong Southbound inflows w w； India domestic flows recovered in June-260710.pdf`

## ✅ 解决方案

### 1. 短路径临时目录
**修复前**: `D:\git\baidu-download\release\dist\temp\` (40字符)
**修复后**: `d:\f` (3字符)
**节省**: 80字符

### 2. 智能文件匹配
**投资银行识别**: Goldman Sachs, Morgan Stanley, J.P. Morgan, Nomura, UBS, Citi等
**数字编号匹配**: 260713, 260710等
**匹配策略**: 精确匹配 + 关键词匹配

### 3. 增强的文件处理
- 文件句柄复制避免路径限制
- 多重备用方案确保成功
- 详细的调试日志

## 🚀 使用指南

### 基础用法
```bash
# 默认模式（上传SFTP）
.\baidu-download.exe --link="分享链接" --folder="目录名"

# 禁用SFTP（不上传）
.\baidu-download.exe --link="分享链接" --folder="目录名" --no-sftp

# 指定提取码（默认0409）
.\baidu-download.exe --link="分享链接" --folder="目录名" --code="1234"
```

### 部署要求
1. **创建临时目录**: `mkdir d:\f`
2. **配置文件**: `.env` (已包含 `TEMP_DIR=d:\f`)
3. **依赖文件**: `BaiduPCS-Go.exe`, `baidu-cookies.txt`
4. **数据库**: MySQL (用于日志记录)

### 环境兼容
- ✅ Windows 10/11
- ✅ Python 3.8+
- ✅ 长文件名支持
- ✅ 特殊字符处理
