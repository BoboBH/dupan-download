# 打包部署测试计划

## 测试目的

验证打包后的程序可以在没有Python环境的Windows电脑上正常运行。

## 测试环境

### 开发机（打包环境）
- Windows 10/11
- Python 3.8+
- 开发工具完整

### 目标机（测试环境）
- Windows 7/8/10/11（无Python环境）
- 干净的系统环境

## 常见问题预案

### 问题1: 程序无法启动

**可能原因**:
- 杀毒软件拦截
- 缺少系统组件
- 文件损坏

**解决方案**:
1. 添加到杀毒软件白名单
2. 安装必要的系统组件
3. 重新复制文件

### 问题2: bypy认证失败

**可能原因**:
- 网络问题
- 百度账号问题
- 认证流程错误

**解决方案**:
1. 检查网络连接
2. 验证百度账号
3. 手动运行 `bypy info`

### 问题3: SFTP连接失败

**可能原因**:
- 配置错误
- 网络问题
- 服务器问题

**解决方案**:
1. 检查 .env 配置
2. 测试网络连接
3. 联系服务器管理员
