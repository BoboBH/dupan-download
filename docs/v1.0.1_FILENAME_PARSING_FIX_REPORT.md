# 🔧 v1.0.1 文件名解析问题修复报告

## 🎯 用户报告的问题

用户发现了文件名映射错误：
```
File downloaded: //260710/Results-260702.pdf -> D:\git\baidu-download\release\dist\temp\901741041_Bobo_Huang123\Bernstein-Asia Quantitative Strategy：Best of Times，Worst of Times：How to invest amidst the Kshaped polarization in Asia-260703.pdf (18312561 bytes)
[INFO] Downloaded:
```

**问题**: 请求下载的文件名和实际下载的文件名不匹配！

## 🔍 根本原因分析

### 1. BaiduPCS-Go 的实际输出格式
通过手动测试，我发现了 BaiduPCS-Go 的 `ls` 命令输出格式：
```
序号    大小      日期    时间     文件名(可能包含空格、中文、特殊字符)
0     7.00MB    2026-07-12 09:12:43  Bernstein-Air Liquide SA（AI.FP）Air Liquide~Key Themes Ahead of 2Q26 Results-260702.pdf
```

### 2. 原来的解析逻辑错误
**问题代码**:
```python
parts = line.split()
if len(parts) >= 4:
    file_name = parts[-1]  # ❌ 只取最后一部分
```

**问题**: 当文件名包含空格时，`split()` 会把文件名也拆开：
```
输入: "0     7.00MB    2026-07-12 09:12:43  Bernstein-Air Liquide SA...Results-260702.pdf"
分割: ['0', '7.00MB', '2026-07-12', '09:12:43', 'Bernstein-Air', 'Liquide', 'SA（AI.FP）', ..., 'Results-260702.pdf']
parts[-1]: "Results-260702.pdf"  # ❌ 只得到最后一部分
```

### 3. 文件名映射错误
- **list_pdf_files 返回**: `//260710/Results-260702.pdf` (不完整)
- **实际下载**: 完整的文件名
- **结果**: 文件名不匹配，下载失败

## ✅ 修复方案

### 1. 正确的解析逻辑
**新代码**:
```python
parts = line.split()
if len(parts) >= 4:
    # 前4个字段是：序号、大小、日期、时间
    # 剩下的部分组合起来就是文件名（可能包含空格）
    file_name = ' '.join(parts[4:])  # ✅ 从第5个部分开始，用空格连接
    file_size_str = parts[1]  # 大小在第2个位置
```

### 2. 修复效果
**示例**:
```
输入: "0     7.00MB    2026-07-12 09:12:43  Bernstein-Air Liquide SA（AI.FP）Air Liquide~Key Themes Ahead of 2Q26 Results-260702.pdf"
分割: ['0', '7.00MB', '2026-07-12', '09:12:43', 'Bernstein-Air', 'Liquide', 'SA（AI.FP）', 'Air', 'Liquide~Key', 'Themes', 'Ahead', 'of', '2Q26', 'Results-260702.pdf']
parts[4:]: ['Bernstein-Air', 'Liquide', 'SA（AI.FP）', ..., 'Results-260702.pdf']
' '.join(parts[4:]): "Bernstein-Air Liquide SA（AI.FP）Air Liquide~Key Themes Ahead of 2Q26 Results-260702.pdf"  # ✅ 完整文件名
```

### 3. 支持复杂文件名
**现在能正确处理**:
- ✅ 中文文件名: `doing？-260702.pdf`
- ✅ 英文文件名: `Results-260702.pdf`
- ✅ 包含空格: `Bernstein-Air Liquide SA...Results-260702.pdf`
- ✅ 包含特殊字符: `：Best of Times，Worst of Times：...-260703.pdf`
- ✅ 长文件名: 超长文件名完整获取
