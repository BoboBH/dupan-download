## **🐛 修复的Bug - Bugs Fixed**

### **✅ Bug #1: 数据库自动创建问题 (已修复)**
- **问题:** 应用程序自动创建新数据库，不使用.env配置
- **影响:** 数据库配置被忽略，创建混乱的数据库实例
- **修复:** 移除CREATE DATABASE语句，确保使用指定数据库
- **文件:** [src/database/models.py](src/database/models.py), [src/database/repository.py](src/database/repository.py)

### **✅ Bug #2: 函数调用参数错误 (已修复)**
- **问题:** `get unexpected keyword argument 'FILE_NAME'`
- **影响:** 程序无法正确调用数据库函数
- **修复:** 函数调用使用正确的小写参数名
- **文件:** [src/processor/file_processor.py](src/processor/file_processor.py)

### **✅ Bug #3: 数据类属性访问错误 (已修复)**
- **问题:** `'FileTransferLog' object has no attribute 'file_name'`
- **影响:** 无法访问数据类属性，导致程序失败
- **修复:** 使用正确的大写属性名 `log.FILE_NAME`
- **文件:** [src/database/repository.py](src/database/repository.py), [main.py](main.py)

### **✅ Bug #4: SQL列名大小写错误 (已修复)**
- **问题:** `Failed to get file log by name and link: 'ID'`
- **影响:** 所有数据库查询操作失败
- **修复:** 所有SQL查询使用小写列名
- **文件:** [src/database/repository.py](src/database/repository.py)

### **✅ Bug #5: 数据库结果字典访问错误 (已修复)**
- **问题:** 数据库返回小写列名，代码尝试使用大写键访问
- **影响:** 无法正确读取数据库查询结果
- **修复:** 字典访问使用小写键 `row['file_name']`
- **文件:** [src/database/repository.py](src/database/repository.py)

### **✅ Bug #6: 数据类实例化参数错误 (已修复)**
- **问题:** `__init__() got an unexpected keyword argument 'id'`
- **影响:** 无法创建数据类对象
- **修复:** 数据类实例化使用大写参数名
- **文件:** [src/database/repository.py](src/database/repository.py)

## **🚀 安装和部署 - Installation and Deployment**

### **1. 系统要求:**
- Windows 10或更高版本
- MySQL 5.7+ 或 MariaDB 10.2+
- 网络连接（用于百度网盘和SFTP）

### **2. 快速安装:**
```bash
# 1. 解压release/dist目录到目标位置

# 2. 配置数据库
编辑 .env 文件，设置数据库连接信息

# 3. 初始化数据库
运行 init_db_auto.bat

# 4. 运行程序
baidu-download.exe --link="..." --folder="..." --code="..."
```

### **3. 配置文件示例 (.env):**
```env
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=baidu_download

# SFTP配置
SFTP_HOST=your_sftp_host
SFTP_PORT=22
SFTP_USER=your_sftp_user
SFTP_PASSWORD=your_sftp_password
SFTP_REMOTE_PATH=/path/to/destination

# 百度网盘配置
BAIDU_COOKIE=your_baidu_cookie

# 临时目录
TEMP_DIR=temp

# 最大重试次数
MAX_RETRIES=3
```

## **✨ 使用示例 - Usage Examples**

### **基本使用:**
```bash
# 下载并上传PDF文件
baidu-download.exe --link="https://pan.baidu.com/s/xxx" --code="1234" --folder="reports"

# 使用默认提取码
baidu-download.exe --link="https://pan.baidu.com/s/xxx" --folder="reports"

# 详细日志模式
baidu-download.exe --link="..." --folder="..." --verbose

# 禁用SFTP上传（仅下载）
baidu-download.exe --link="..." --folder="..." --no-sftp
```

```bash
# 测试配置
baidu-download.exe --link="..." --folder="..." --dry-run

# 使用自定义配置文件
baidu-download.exe --link="..." --folder="..." --config="custom.env"
```
