## 🚀 使用方法

1. **配置环境**
   ```
   copy .env.example .env
   编辑 .env 文件，填入你的配置
   ```

2. **初始化数据库**
   ```
   init_db_auto.bat   (Windows)
   或
   init_db_auto.sh    (Linux/Mac)
   ```

3. **运行程序**
   ```
   baidu-download.exe --link "分享链接" --code "提取码" --folder "目录"
   ```

## 📋 文件说明

- `baidu-download.exe` - 主程序 (11MB，包含所有功能)
- `BaiduPCS-Go.exe` - 百度网盘客户端
- `.env` - 配置文件模板
- `VERSION` - 版本信息
- `init_db_auto.bat/sh` - 数据库初始化脚本

## 🔧 配置说明

编辑 `.env` 文件：

```
# 百度网盘配置
BD_SHARE_LINK=你的分享链接
BD_CODE=提取码

# SFTP配置
SFTP_HOST=sftp服务器地址
SFTP_PORT=22
SFTP_USER=用户名
SFTP_PASSWORD=密码
REMOTE_PATH=/home/username/files

# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=密码
DB_NAME=baidu_download
```
