# 百度网盘下载工具 - 部署包说明

## 📦 部署包内容

本部署包是一个**完全独立**的程序包，可以在任何 Windows 64位机器上运行。

### 包含文件

```
dupan-download-windows-clean/
├── pan-download.exe          # 主程序（18 MB，包含所有依赖）
├── .env.example              # 配置文件模板
├── README.md                 # 项目说明
├── INSTALL_GUIDE.md          # 详细安装指南
├── 快速开始.bat              # 快速配置向导
├── 验证安装.bat              # 部署验证脚本
└── 部署说明.md              # 本文件
```

## 🚀 快速部署

### 方法一：直接使用（推荐）

1. **解压到目标位置**
   ```
   解压到：C:\Program Files\dupan-download\
   或：D:\tools\dupan-download\
   或任何你想要的位置
   ```

2. **运行快速开始向导**
   ```
   双击：快速开始.bat
   ```

3. **完成配置**
   - 编辑 `.env` 文件（可选）
   - 配置百度网盘认证
   - 测试功能

### 方法二：手动配置

1. **解压部署包**

2. **创建配置文件**
   ```cmd
   copy .env.example .env
   notepad .env
   ```

3. **配置 SFTP（可选）**
   如需自动上传功能，编辑 `.env` 文件：
   ```env
   SFTP_HOST=your.server.com
   SFTP_PORT=22
   SFTP_USERNAME=your_username
   SFTP_PASSWORD=your_password
   SFTP_REMOTE_PATH=/remote/path
   ```

4. **配置百度网盘**
   ```cmd
   pan-download.exe --setup-bypy
   ```

5. **测试功能**
   ```cmd
   pan-download.exe --test-config
   pan-download.exe apps/bypy/test_pdf --keep-temp
   ```

## 📖 使用示例

### 基本下载
```cmd
# 下载到临时目录（自动清理）
pan-download.exe apps/bypy/my_folder

# 下载并保留文件
pan-download.exe apps/bypy/my_folder --keep-temp
```

### 下载并上传
```cmd
# 下载后自动上传到 SFTP
pan-download.exe apps/bypy/my_folder --upload-sftp
```

### 高级选项
```cmd
# 指定本地目录
pan-download.exe apps/bypy/my_folder --local-dir "D:\Downloads"

# 详细输出
pan-download.exe apps/bypy/my_folder --verbose

# 测试配置
pan-download.exe --test-config
```

## 🔧 故障排除

### 问题：程序无法启动

**解决方案：**
1. 确认是 64位 Windows 系统
2. 以管理员身份运行
3. 检查杀毒软件是否阻止
4. 在命令提示符中运行查看错误：
   ```cmd
   pan-download.exe --help
   ```

### 问题：百度网盘认证失败

**解决方案：**
```cmd
# 重新运行认证向导
pan-download.exe --setup-bypy

# 或手动配置（需要有 Python 的机器）
pip install bypy
bypy info
# 然后复制认证文件到目标机器
```

### 问题：SFTP 连接失败

**解决方案：**
1. 检查 `.env` 文件中的 SFTP 配置
2. 测试网络连接：
   ```cmd
   ping your.server.com
   ```
3. 验证用户名和密码

## 📋 配置文件说明

### .env 文件结构

```env
# 百度网盘认证（自动管理，无需手动配置）
# 认证信息保存在 ~/.bypy/ 目录中

# SFTP 服务器配置（可选，如需上传功能）
SFTP_HOST=your.server.com
SFTP_PORT=22
SFTP_USERNAME=your_username
SFTP_PASSWORD=your_password
SFTP_REMOTE_PATH=/remote/path

# 性能配置（可选）
MAX_RETRIES=3
CONNECT_TIMEOUT=30
TRANSFER_TIMEOUT=300
```

## 🔒 安全建议

1. **保护配置文件**
   - `.env` 包含敏感信息，不要分享
   - 不要将 `.env` 提交到版本控制

2. **网络安全**
   - 在受信任的网络中使用
   - 使用 SFTP 而非普通 FTP
   - 考虑使用 SSH 密钥认证

3. **文件安全**
   - 定期清理临时文件
   - 验证下载文件的完整性
